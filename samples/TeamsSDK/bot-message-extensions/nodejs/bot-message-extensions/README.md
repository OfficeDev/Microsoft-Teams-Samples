# Bot Message Extensions - Node.js (TypeScript)

This sample demonstrates a search-based messaging extension in Microsoft Teams that allows users to search for  Wikipedia articles.

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS version recommended)

## Run the sample

1. Navigate to this directory:
   ```bash
   cd nodejs/bot-message-extensions
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the bot:
   ```bash
   npm start
   ```

The bot will start listening on `http://localhost:3978`.

Refer to the main [README.md](../../README.md) to interact with your bot in the agentsplayground or in Teams.

# Bot Task Modules - TypeScript

This sample demonstrates Task Modules for Microsoft Teams using TypeScript.

## Prerequisites

- [Node.js](https://nodejs.org) version 20.x or higher
- npm (included with Node.js)

## Run the sample

1. Navigate to this directory:
   ```bash
   cd nodejs/bot-task-modules
   ```

2. Install dependencies and run the bot:
   ```bash
   npm install
   npm start
   ```

The bot will start listening on `http://localhost:3978`.

Refer to the main [README.md](../../README.md) to interact with your bot in the agentsplayground or in Teams.

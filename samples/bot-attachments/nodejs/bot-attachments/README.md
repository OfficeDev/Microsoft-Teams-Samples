# Bot Attachments - TypeScript

This sample demonstrates file handling and attachments for Microsoft Teams using Node.js and TypeScript.

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS version recommended)

## Run the sample

1. Navigate to this directory:
   ```bash
   cd nodejs/bot-cards/bot-attachments
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the bot:
   ```bash
   npm start
   ```

The bot will start listening on `http://localhost:3978`.

Refer to the main [README.md](../../README.md) to interact with your bot in the agentsplayground or in Teams.

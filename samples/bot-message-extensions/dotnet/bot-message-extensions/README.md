# Bot Message Extensions - .NET (C#)

This sample demonstrates a search-based messaging extension in Microsoft Teams that allows users to search for NuGet packages and Wikipedia articles directly within the compose area.

## Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)

## Run the sample

1. Navigate to this directory:
   ```bash
   cd dotnet/BotMessageExtensions
   ```

2. Restore dependencies and run:
   ```bash
   dotnet run
   ```

The bot will start listening on `http://localhost:3978`.

Refer to the main [README.md](../../README.md) to interact with your bot in the Microsoft 365 Agents Playground or in Teams.

Microsoft Teams Dialogs (referred as task modules in TeamsJS v1.x) Demo

A dialog (referred as task modules in TeamsJS v1.x) allows Teams app developers to create one or more custom, modal experiences with custom code or Adaptive cards for their users, particularly useful for initiating and/or completing tasks using a GUI that does not litter a Teams conversation with superfluous, intermediate task completion information.
